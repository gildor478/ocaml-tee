This is the README file for the ocaml-tee distribution.

Redirect command output to a file

(C) 2009 Sylvain Le Gall

See the files INSTALL.txt for building and installation instructions.

Home page: http://ocaml-tee.forge.ocamlcore.org

Library and command line utility like POSIX tee command. It allows to
redirect stderr/stdout of a command to a file. Unlike POSIX tee, the file has
a specific format to differentiate time/stdout/stderr.

See the file COPYING.txt for copying conditions.
